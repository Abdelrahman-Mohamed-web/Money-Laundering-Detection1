=====================================================================
  MONEY LAUNDERING DETECTION - macOS INSTRUCTIONS
=====================================================================
  Step-by-step guide for running the entire project on macOS.
  All commands use the Terminal app.
=====================================================================

--------------------------------------------------------------------
  0. WHAT YOU NEED BEFORE YOU START
--------------------------------------------------------------------
 (a) Python 3.10 or newer. 
 (b) The project folder "money-laundering-detection" (unzipped).
 (c) (Optional) The Kaggle CSV "AML_Fraud_full.csv".

--------------------------------------------------------------------
  1. OPEN TERMINAL IN THE PROJECT FOLDER
--------------------------------------------------------------------
  $ cd ~/Downloads/money-laundering-detection    

--------------------------------------------------------------------
  2. CREATE AND ACTIVATE A VIRTUAL ENVIRONMENT (venv)
--------------------------------------------------------------------
      $ python3 -m venv .venv
      $ source .venv/bin/activate

--------------------------------------------------------------------
  3. INSTALL THE PYTHON DEPENDENCIES
--------------------------------------------------------------------
      $ pip install --upgrade pip
      $ pip install -r requirements.txt

--------------------------------------------------------------------
  4. (OPTIONAL) DOWNLOAD THE KAGGLE DATASET
--------------------------------------------------------------------
  Put the file at exactly: data/raw/AML_Fraud_full.csv

--------------------------------------------------------------------
  5. TRAIN THE MODEL
--------------------------------------------------------------------
      $ python -m src.training.train

--------------------------------------------------------------------
  6. MAKE PREDICTIONS ON NEW TRANSACTIONS
--------------------------------------------------------------------
  Single transaction:
      $ python -m src.predict --amount 1000000 --type "TRANSFER"

  Expected output (JSON):
      {
        "amount": 1000000,
        "type": "TRANSFER",
        "prediction": "fraud",
        "probability": 0.97
      }

  Batch CSV:
      $ python -m src.predict --csv data/raw/AML_Fraud_full.csv

--------------------------------------------------------------------
  7. RUN THE UNIT TESTS
--------------------------------------------------------------------
      $ pytest -v
=====================================================================
